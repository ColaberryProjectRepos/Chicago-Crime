Master Items and Drill Down Functionality 8

We need to create calculated measures and Dimensions to be able to calculate with and without filters, percentages, etc for our reports.

Master Items are dimensions, measures and visualizations that have been saved in the Asset panel in a Qlik Sense app.  During the development process, Master Items can be used to create visualizations and once the app is published, the Master Items are available in the Library panel.  Drill downs are dimensions or measures which can allow  a user to explore different layers of data.  Below is how we create a drill down functionality within our dashboard. This drill down will be saved in master items for later use.

below are some examples of how to create master items in Qlik Sense

You Create a Measure by clicking "Master Items" and then " Measures" or “Dimensions”


For Dimensions we Created SideCommunity, YearsMonthWeeks and ZipCommunity Drill Down.

For Measures we Created VCR(Violent Crime Rate) and Avg Violent crime, USCR (Unsolved Crime Rate) and Avg unsolved crime.

Then you need to copy paste the expression for the measures below. Create a measure for each expression from a to d.



a)     VCR = Count({$<Violent={'1'}>}Violent)/Count(Violent)

b)     USCR = Count({<[Arrest] = {'false'}>} [Arrest]) / COUNT(ID)

c)     AvgVCrime = AVG(TOTAL Violent)

d)     AvgUSCrime = (=Count({<[Arrest] = {'false'}>} [Arrest]) / COUNT(ID))

Reusing dimensions with master dimensions

When you are working with an unpublished app, you can create master dimensions so that they can be reused. Users of a published app will have access to the master dimensions, but will not be able to modify them.

You can create a master dimension in different ways.

  • Creating a master dimension from a field
  • Creating a master dimension from the assets panel
  • Creating a master dimension from the data model viewer

  • You can also create a master dimension based on a dimension created in a visualization.

    Click Master item> Create new under the dimension in the property panel of a visualization.

    The Create new dimensions dialog opens, with the dimension value from the property panel in Field.
